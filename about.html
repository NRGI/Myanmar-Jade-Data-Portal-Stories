EITI
Extractive Industries Transparency Initiative.

EITI Independent Administrator (IA)
An independent consultant who prepares the EITI report and conducts the reconciliation exercise on behalf of an EITI National Coordinator.

EITI Multi-Stakeholder Group (MSG)
A group made up of government, company, and civil society representatives that oversee the EITI implementation in a country. The MSGdevelops the country Workplan, the production of the EITI Report and ensures that the EITI contributes to public debate.

EITI National Coordinator
The National Coordinator leads the EITI process on a day-to-day basis in an implementing country and heads the national EITI secretariat. The National Coordinator may be part of or chairs the multi-stakeholder group.

EITI Reconciliation Scope
Extractives companies have to go through the reconciliation exercise if the revenues obtained from them by the government exceeds a certain cut-off called the materiality threshold. The companies that exceed this cut-off are included in the reconciliation scope, whereas others are not. For the gems and jade sector in Myanmar, the materiailty threshold was set at MMK 1 billion in the 2014-15 and 2015-16 EITI reports, and MMK 10 billion in the 2013-14 EITI report.

EITI Reconciliation Sheet
Forms used be the EITI independent administrator that that is used to reconcile revenue data from companies and the government.

EITI Summary Sheet
Excel sheet prepared by the EITI independent administrator that summarises the reconciled revenue data from the main EITI report.

Jupyter Notebook
An open-source web application that allows you to create and share documents that contain live code, equations, visualizations and narrative text. Uses include: data cleaning and transformation, numerical simulation, statistical modeling, data visualization, machine learning, etc. Python Jupyter Notebooks are Jupyter Notebooks which contain code written in the Python programming language. .csv
Comma-Seperated Values. A data format for storing tabular data in a open and machine readable format. You can save a spreadsheet as a ".csv" file using spreadsheet software such as Excel or Google Sheets.

.ipynb
File format for Jupyter Notebooks.

.tar.gz
Open-sourced compressed file format. In this website, it is used for storing data cleaining projects produced in OpenRefine.

.xlsx
Microft Excel's default data format. Since it is a proprietary file format, data stored in ".xlsx" files is not considered the best way to distribute open data. You can save a spreadsheet as a ".csv" file using spreadsheet software such as Excel or Google Sheets.
